# Datatypes

Datatypes's first character are always uppercase.

## Int

Same as php int

```thp
Int age = 32
Int red = 0xff0000
Int char_code = 0b01000110
```


## Float

Same as php float


```thp
Float pi = 3.141592
```


## String

Same as php string

```thp
String name = "Rose"
```


## Bool

Same as php bool

```thp
Bool is_true = true
```
