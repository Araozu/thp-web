# Hello, world!

Create a file named `hello.thp` with the contents:

```thp
print("Hello, world!")
```

Then run `thp hello.thp`

## Try

<InteractiveCode 
    code={`
    val x = 322
    var y = 322

    fun f(Int a, Int b) {
        print("hello {a} {b}")
    }

    f(x, y)
    `}
    steps={`[
        [["line", 1]],
        [
            ["line", 2],
            ["set", "x", "322"]

        ],
        [["line", 4]],
        [["line", 5]],
        [
            ["out", "gaaaa 322 322"],
            ["line", 6]
        ],
        [["line", 8]],
        [["line", 0]]
    ]`}
></InteractiveCode>
