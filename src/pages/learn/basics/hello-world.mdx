# Hello, world!

Create a file named `hello.thp` with the contents:

```thp
print("Hello, world!")
```

Then run `thp hello.thp`

## Try

```thp
val x = 322
val y = 322

fun f(Int x, Int y) {
    print("gaaaa {x} {y}")
}

f(x, y)
```
